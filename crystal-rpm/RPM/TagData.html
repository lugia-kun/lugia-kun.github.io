    def pos=(idx : Int) #

    Sets current index of TagData.

    This is for internal use.

    #value_no_array and #value_array ignores this property, they always return the value (of non-array data) or the array of the all values.

    Getting position is not supported, because it will be random if you use multiple Iterators at once.
