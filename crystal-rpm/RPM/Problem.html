    ======================================================================== type pkg_nevr alt_nevr str number

    BADARCH target pkg (unused) arch name (unused) BADOS target pkg (unused) os name (unused) PKG_INSTALLED target pkg (unused) (unused) (unused) BADRELOCATE target pkg (unused) relocated path (unused) NF_CONFLICT target pkg conflicting pkg file path (unused) FILE_CONFLICT installed pkg conflicting pkg file path (unused) OLD_PACKAGE target pkg installed pkg (unused) (unused) DISKSPACE target pkg (unused) file system number bytes DISKNODES target pkg (unused) file system number inodes REQUIRES required pkg pkg requires+2 (unused) 0 = installed CONFLICTS conflict pkg pkg conflicted+2 (unused) 0 = installed (OBSOLETES) obsoletes pkg pkg obsoleted+2 (unused) 0 = installed (VERIFY) ========================================================================

    OBSOLETES is not defined in RPM 4.8, but applies same translation to REQUIRES or CONFLICTS.

    No translation applies for VERIFY.
